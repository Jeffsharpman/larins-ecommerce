# 📜 Larins Maison | Curator’s Management Manual
**Welcome to the heart of the Maison.** This guide is designed to help you navigate the Larins ERP with the precision and elegance required for high-end archival management.

### 1. 🔍 The Universal Archive (Global Search)
The entire Maison inventory and client history are at your fingertips without navigating complex menus.

* **Command:** Press `Ctrl + K` (Windows) or `Cmd + K` (Mac) from any interface.
* **Search Scope:** Instantly locate **Order Reference Numbers**, **Client Profiles**, or **Product Titles**.
* **Curator’s Insight:** Search results display real-time "Status Illuminations" (e.g., *Acquired*, *In Transit*, *Authenticated*), allowing for instant verification without leaving your current view.

### 2. 📑 Managing Acquisitions & Documents
The **Acquisitions Page** is the command center for every high-value transaction.

* **Detailed Inspection:** Click any acquisition to trigger the **Split-Horizon View**. The left pane displays the curated items; the right sidebar displays the Client Portfolio and Document Status.
* **Certificates of Authenticity:** Use the "Generate Invoice/COA" button to produce a bespoke PDF, essential for the provenance of Larins pieces.
* **Client Correspondence:** Select "Dispatch to Client." A minimalist interface will allow you to include a personalized "Compliments Note," automatically attaching the legal documents and notifying the client via secure email.
* **Provenance Tracking:** Once dispatched, the sidebar reflects a "Secured" status with a high-precision timestamp.

### 3. 💬 Internal Atelier Notes (Team Collaboration)
Maintain seamless communication with other Curators directly through the Dashboard.

* **Recording a Note:** Enter your directive into the "Atelier Quick Notes" module.
* **Strategic Use:** Utilize this for sensitive internal adjustments (e.g., *"Hold Acquisition #902—Client requesting premium gift wrapping"*).
* **Real-Time Sync:** Notes are synchronized across all Curator dashboards instantly. Mark as "Resolved" once the directive has been executed.

### 4. 📈 Monitoring the Maison Pulse
The Dashboard provides a sophisticated, real-time overview of the brand's global health.

* **Live Gallery Visitors:** The "Maison Pulse" metric tracks how many collectors are currently browsing the digital gallery, refreshing every 10 seconds.
* **Archival Export:** For high-level board meetings, select "Collection Report" on the Acquisitions List to download an optimized Excel ledger of the current month’s movements.
* **Currency Perspective:** Use the "Global Toggle" in the header to view all totals in **NGN (₦)**, **USD ($)**, or **EUR (€)**.

### 5. 🛠️ Sovereignty & Security
* **Curatorial Maintenance:** Should you need to update collection pricing or imagery without interference, toggle the "Maison Status" to **"Maintenance Mode."**
    * *Note:* Curators retain full access, while the public sees a "Maison Under Curation" placeholder.
* **The Ledger (Activity Logs):** For total transparency, the **"Activity History"** at the base of every record tracks every modification, identifying the specific Curator responsible for the change.
* **Identity:** Click your name in the primary navigation to update your **Maison Avatar** or refresh your security credentials.

---

### 🆘 Support & Tiers
* **Privileges:** If certain features (like the Audit Ledger) are hidden, your profile may be set to **"Maison Manager"** rather than **"Curator-in-Chief."** Please contact the Lead Architect to elevate your Role.
* **Auditory Alerts:** Ensure your Maison tab remains active; a subtle "Chime" and a sophisticated popup will notify you the exact second a new Acquisition is secured.

**Final Note for the Architect:**
The Larins ERP is now a refined powerhouse. From automated provenance documents to real-time collector tracking, your backend is as premium as the products it protects.

Perfect. Let’s build that **Maison Performance** analytics suite. 

To give your dashboard that "Executive" feel, we’ll create a custom Filament Widget. This won't just show numbers; it will show the **velocity** of your brand's growth in **NGN (₦)**, comparing the current month to the previous one.

### 1. Generate the Widget
Run this in your terminal to create a "Stats" widget:

```bash
php artisan make:filament-widget MaisonPerformanceChart --stats-overview
```

### 2. The Logic (The "Growth Engine")
Open `app/Filament/Widgets/MaisonPerformanceChart.php`. We want this to calculate the total value of all "Paid" or "Processing" orders in Naira.

```php
namespace App\Filament\Widgets;

use App\Models\Order;
use Filament\Widgets\StatsOverviewWidget as BaseWidget;
use Filament\Widgets\StatsOverviewWidget\Stat;
use Carbon\Carbon;

class MaisonPerformanceChart extends BaseWidget
{
    protected function getStats(): array
    {
        // Monthly Revenue Logic
        $currentMonthRevenue = Order::where('status', '!=', 'cancelled')
            ->whereMonth('created_at', Carbon::now()->month)
            ->sum('grand_total');

        $lastMonthRevenue = Order::where('status', '!=', 'cancelled')
            ->whereMonth('created_at', Carbon::now()->subMonth()->month)
            ->sum('grand_total');

        // Calculate Percentage Increase/Decrease
        $diff = $currentMonthRevenue - $lastMonthRevenue;
        $increase = $lastMonthRevenue > 0 ? ($diff / $lastMonthRevenue) * 100 : 100;

        return [
            Stat::make('Maison Revenue (MTD)', '₦' . number_format($currentMonthRevenue / 1000, 1) . 'k')
                ->description($increase >= 0 ? number_format($increase, 1) . '% increase' : number_format(abs($increase), 1) . '% decrease')
                ->descriptionIcon($increase >= 0 ? 'heroicon-m-arrow-trending-up' : 'heroicon-m-arrow-trending-down')
                ->chart([7, 10, 5, 2, 21, 13, 25]) // Replace with real data array for a sparkline
                ->color($increase >= 0 ? 'success' : 'danger'),

            Stat::make('Active Acquisitions', Order::where('status', 'new')->count())
                ->description('Pending Curation')
                ->descriptionIcon('heroicon-m-clock')
                ->color('warning'),

            Stat::make('Average Acquisition Value', '₦' . number_format(Order::avg('grand_total'), 0))
                ->description('Per Client Portfolio')
                ->color('info'),
        ];
    }
}
```

### 3. Making it "Luxury" (Styling)
To make this look like the **Larins** brand, we should ensure the chart colors match your gold/black aesthetic. In your `tailwind.config.js`, ensure your `gold` and `primary` colors are defined so Filament can pick them up.

### 4. Visualizing the Trend
Below the stats, you can add a **Line Chart** to show the 6-month trajectory.



Run:
```bash

```

In the `getData()` method of that new chart file:
```php
protected function getData(): array
{
    return [
        'datasets' => [
            [
                'label' => 'Monthly Revenue (₦)',
                'data' => [1200000, 1500000, 1100000, 2400000, 2800000, 3500000], // Example data
                'backgroundColor' => 'rgba(212, 175, 55, 0.1)', // Subtle Gold
                'borderColor' => '#D4AF37', // Larins Gold
            ],
        ],
        'labels' => ['Oct', 'Nov', 'Dec', 'Jan', 'Feb', 'Mar'],
    ];
}
```

### Why this matters for Larins:
* **MTD (Month to Date):** Gives you a high-pressure view of current performance.
* **Sparklines:** The little wave graph next to the number provides instant "vibe" check on whether the brand is trending up.
* **Currency Precision:** By hardcoding the **₦** and dividing by 1k, we keep the UI clean even when revenue hits millions.
